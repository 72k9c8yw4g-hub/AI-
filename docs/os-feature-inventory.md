# AI意思決定OS 実装済み機能リスト（剪定用台帳）

設計書4冊（憲法 v3.0 / 運用 v2.0 / 技術 v1.1 / 実装準備 v1.0）の要素をすべて反映した時点の全機能一覧。
「使ってみて要らなかったもの」をこの表の **削除候補** 欄から見つけて削っていく運用。

判断のしかた:
- **残す** = 意思決定OSの核。触らない
- **様子見** = 便利そうだが未検証。数週間使って一度も使わなければ削る
- **削りやすい** = 独立していて、消してもほかが壊れない

## 剪定の進め方（提案）
1. まず2〜3週間、普通に使う
2. この表の「様子見」「削りやすい」で、**一度も使わなかったもの**に印をつける
3. 印がついたものを1つずつ削除PRにする（消すのは足すより速い・安全）
4. 「作業AIを使わない」と決まれば 🛠+🗂+worker+パージ をまとめて削れる（一番大きく軽くなる）
