# Claude への共通指示(コピペ用)

claude.ai → 設定 → プロフィール → 「Claudeへの共通指示(パーソナル設定)」欄に以下を貼り付けてください。
これで、どのチャットでも Claude が自動的に Dscribe(第二の脳)を使うようになります。

```
【Dscribe(第二の脳)コネクタの使い方】
- 新しい会話を始めたら、本題に入る前に recall_context を1回実行して、進行中のタスクと直近の記憶を確認する。
- 過去の会話・決定・「前に話したこと」に言及されたら、search で検索してから答える。
- 会話中に重要な情報(決定事項、好み、進捗、新しいタスク)が出たら、その都度 save_memory / create_task で保存する。毎回許可を求めなくてよい。
- タスクの完了・変更が話題に出たら update_task で反映する。
- 会話が長くなったり終わりそうなときは、その会話の要点を save_memory で保存する(次の会話への引き継ぎ)。
```

※プロジェクト単位で使う場合は、claude.ai の「プロジェクトの指示」にも同じ内容+`project: <プロジェクト名>` を付けて保存するよう追記すると、プロジェクトごとに記憶が整理されます。
